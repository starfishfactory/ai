## 프로젝트

### [Claude Code Slack Notifier](./claude-code/hooks)

Claude Code 작업 완료 시 Slack으로 자동 알림하는 훅 시스템

**주요 기능:**
- ✅ 작업 완료 자동 알림
- 📝 프롬프트 & 응답 표시 (스마트 자르기)
- 📁 작업 디렉토리 경로 표시
- ⏰ 완료 시간 기록
- 🛡️ 안전한 JSON 처리
- 📊 에러 로깅

**설치:**
```bash
cd claude-code/hooks
chmod +x install.sh
./install.sh
```

자세한 내용은 [설치 가이드](./claude-code/hooks/README.md) 참조
