# SWOT 분석: $ARGUMENTS

## 분석 요청
**$ARGUMENTS**에 대해 SWOT 분석 및 TOWS 전략을 도출하세요.

## 분석 항목
| 구분 | 내용 |
|------|------|
| **S**trengths | 경쟁 우위, 핵심 역량, 자산 |
| **W**eaknesses | 경쟁 열위, 개선 필요 영역 |
| **O**pportunities | 시장 기회, 트렌드, 규제 완화 |
| **T**hreats | 경쟁 심화, 규제 강화, 기술 변화 |

## 출력 형식

```
# $ARGUMENTS SWOT 분석

## SWOT 매트릭스
| | 긍정적 | 부정적 |
|---|--------|--------|
| **내부** | **S**: 1. 2. 3. | **W**: 1. 2. 3. |
| **외부** | **O**: 1. 2. 3. | **T**: 1. 2. 3. |

## TOWS 전략
| | S 활용 | W 보완 |
|---|--------|--------|
| **O** | SO: | WO: |
| **T** | ST: | WT: |

## 핵심 전략 권고
1.
2.

## 출처
-
```

**각 S/W/O/T 항목 3개씩, 1줄로 간결하게.**
