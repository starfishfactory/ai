## 3. 새로 발견된 문제 🆕

### 3.1 한국어/영어 혼용 여전함

```markdown
| 요인 | 현황 (1줄) | 트렌드 | 영향도 |
|------|-----------|--------|--------|
| P | | ↑/→/↓ | 상/중/하 |
```

- 컬럼명: 한국어
- 요인명: 영문 약자 (P, E, S, T, E, L)
- 트렌드: 특수문자
- 영향도: 한국어

### 3.2 skills 디렉토리 구조 의문

Claude Code 문서에 따르면:
```
skills/
  skill-name/
    SKILL.md (필수)
```

**현재 구조**:
```
skills/
  porter-five-forces.md
  pestel-framework.md
  ...
```

→ `SKILL.md` 형식이 아님. 실제로 로드될지 불명확.

### 3.3 명령어에서 스킬 참조 구문 없음

간소화된 `pestel.md`:
```markdown
# PESTEL 분석: $ARGUMENTS

## 분석 요청
**$ARGUMENTS**에 대해 PESTEL 거시환경 분석을 수행하세요.
```

- 이전: `skills/pestel-framework.md`의 PESTEL 프레임워크를 기반으로...
- 현재: 스킬 참조 구문 **삭제됨**
- 스킬 파일이 실제로 사용되는지 더욱 불명확해짐

### 3.4 데이터 소스 문제 여전

| 항목 | 상태 |
|------|------|
| 실시간 데이터 API | ❌ 없음 |
| 금융 데이터 | ❌ 없음 |
| 산업 보고서 | ❌ 없음 |
| WebSearch 의존 | ⚠️ 부정확 가능성 |
