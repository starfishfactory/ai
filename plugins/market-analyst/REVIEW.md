| skills 디렉토리 구조가 표준과 일치하는가? | ❓ 불확실 |

### 3.2 skills 디렉토리 구조 의문

Claude Code 문서 표준:
```
skills/
  skill-name/
    SKILL.md (필수)
```

현재 구조:
```
skills/
  porter-five-forces.md
  pestel-framework.md
  ...
```

→ `SKILL.md` 형식이 아님. 실제 로드 여부 불명확.
