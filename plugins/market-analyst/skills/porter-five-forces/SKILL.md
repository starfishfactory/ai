# Porter's Five Forces

## 출처
- **창시자**: Michael E. Porter
- **기관**: Harvard Business School
- **연도**: 1979년
- **원문**: "How Competitive Forces Shape Strategy", HBR

## 5가지 경쟁요인

```
        [신규 진입 위협]
              ↓
[공급자] ← [기존 경쟁] → [구매자]
              ↑
        [대체재 위협]
```

| 요인 | 핵심 질문 |
|------|----------|
| 신규 진입 | 진입장벽 높이는? |
| 공급자 교섭력 | 공급자 집중도? 대체 공급원? |
| 구매자 교섭력 | 구매자 집중도? 전환비용? |
| 대체재 | 가격/성능 비율? 전환 트렌드? |
| 기존 경쟁 | 경쟁자 수? 성장률? 차별화? |

## 산업 매력도
5가지 요인 강도 종합 → 높음/중간/낮음
