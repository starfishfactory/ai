# 시장규모 추정 (TAM/SAM/SOM)

## 정의

| 용어 | 정의 |
|------|------|
| **TAM** | Total Addressable Market (전체 시장) |
| **SAM** | Serviceable Addressable Market (접근 가능) |
| **SOM** | Serviceable Obtainable Market (확보 가능) |

## 추정 방법

### Top-Down
```
TAM = 산업규모 × 관련비율
```

### Bottom-Up
```
TAM = 잠재고객수 × 연간구매액
```

### 산출 체인
```
TAM → SAM (지역/세그먼트 적용) → SOM (점유율 적용)
```

## CAGR
```
CAGR = (최종년도/기준년도)^(1/n) - 1
```
