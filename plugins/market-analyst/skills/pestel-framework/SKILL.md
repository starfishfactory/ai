# PESTEL Analysis Framework

## Origin
- **Creator**: Francis J. Aguilar
- **Institution**: Harvard Business School
- **Year**: 1967
- **Source**: "Scanning the Business Environment"

## 6 Factors
| Factor | Key Items |
|--------|----------|
| **P**olitical | Government policy, taxation, trade regulation |
| **E**conomic | GDP, inflation, interest rates, exchange rates |
| **S**ocial | Demographics, lifestyle, values |
| **T**echnological | Innovation, digital transformation, R&D |
| **E**nvironmental | Climate change, ESG, environmental regulation |
| **L**egal | Industry regulation, consumer protection, data protection |

## Analysis Flow
```
Macro-environment scan -> Trend identification -> Impact assessment -> Implications
```

## Linkage
PESTEL -> feeds into SWOT O/T
