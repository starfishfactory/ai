# SWOT 분석 프레임워크

## 출처
- **창시자**: Albert S. Humphrey
- **기관**: Stanford Research Institute
- **연도**: 1960-1970년대

## 구조

```
           | Helpful    | Harmful
-----------+------------+-----------
Internal   | Strengths  | Weaknesses
External   | Opportunities | Threats
```

## TOWS 전략 매트릭스

| | S 활용 | W 보완 |
|---|--------|--------|
| **O** | SO: 강점으로 기회 활용 | WO: 약점 보완하여 기회 활용 |
| **T** | ST: 강점으로 위협 대응 | WT: 약점/위협 최소화 |

## 연계
- PESTEL → O/T 도출
- Porter → 경쟁 O/T
- Value Chain → S/W 도출
