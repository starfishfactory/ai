# SWOT 분석 프레임워크

## 개요

| 항목 | 내용 |
|------|------|
| **창시자** | Albert S. Humphrey |
| **연구기관** | Stanford Research Institute (SRI) |
| **발표** | 1960-1970년대 |
| **원래 명칭** | SOFT → SWOT (1964년 취리히 세미나) |
| **연구팀** | Marion Dosher, Otis Benepe, Albert Humphrey, Robert Stewart, Birger Lie |

## 이론적 배경

SWOT 분석은 **내부 역량**과 **외부 환경**을 종합적으로 평가하여 전략적 의사결정을 지원하는 프레임워크입니다.

### 연구 배경
> Fortune 500 기업의 기업 계획(Corporate Planning) 실패 원인을 규명하기 위해 SRI에서 1960-1970년 수행한 연구에서 탄생

### 명칭 변화
```
SOFT (Stewart, 1964)
- Satisfactory: 현재 좋은 것
- Opportunity: 미래에 좋은 것
- Fault: 현재 나쁜 것
- Threat: 미래에 나쁜 것

SWOT (Urick & Orr, 1964)
- Strengths, Weaknesses, Opportunities, Threats
```

## SWOT 매트릭스 구조

```
              │ 긍정적 (Helpful)  │ 부정적 (Harmful)
──────────────┼──────────────────┼──────────────────
내부 (Internal)│   Strengths      │   Weaknesses
              │   강점            │   약점
──────────────┼──────────────────┼──────────────────
외부 (External)│   Opportunities  │   Threats
              │   기회            │   위협
```

## 4가지 구성요소

### 1. Strengths (강점)

**정의**: 조직이 경쟁사 대비 우위를 가지는 내부 역량

| 분석 영역 | 세부 항목 |
|----------|----------|
| 자원 | 재무, 인력, 기술, 브랜드 |
| 역량 | 핵심 역량, 차별화 요소 |
| 자산 | 특허, 시설, 네트워크 |
| 성과 | 시장점유율, 수익성 |

**분석 질문**:
- 경쟁사 대비 우위 요소는?
- 고객이 우리를 선택하는 이유는?
- 우리만의 차별화된 역량은?
- 보유한 핵심 자원은?

### 2. Weaknesses (약점)

**정의**: 조직이 경쟁사 대비 열위에 있는 내부 요인

| 분석 영역 | 세부 항목 |
|----------|----------|
| 자원 부족 | 자본, 인력, 기술 |
| 역량 미흡 | 경쟁력 취약 분야 |
| 구조적 문제 | 비효율, 조직 문화 |
| 성과 부진 | 저성장, 저수익 영역 |

**분석 질문**:
- 개선이 필요한 영역은?
- 경쟁사 대비 열위 요소는?
- 고객 불만 요인은?
- 피해야 할 것은?

### 3. Opportunities (기회)

**정의**: 조직에 유리하게 활용할 수 있는 외부 환경 요인

| 분석 영역 | 세부 항목 |
|----------|----------|
| 시장 | 시장 성장, 신규 세그먼트 |
| 기술 | 기술 혁신, 디지털 전환 |
| 규제 | 규제 완화, 정부 지원 |
| 경쟁 | 경쟁자 약화, 인수 기회 |

**분석 질문**:
- 시장의 성장 기회는?
- 활용 가능한 트렌드는?
- 경쟁사가 놓치고 있는 것은?
- 규제/기술 변화의 기회는?

### 4. Threats (위협)

**정의**: 조직에 부정적 영향을 줄 수 있는 외부 환경 요인

| 분석 영역 | 세부 항목 |
|----------|----------|
| 경쟁 | 신규 진입, 경쟁 심화 |
| 시장 | 시장 축소, 수요 변화 |
| 환경 | 경기 침체, 규제 강화 |
| 기술 | 기술 진부화, 대체재 |

**분석 질문**:
- 경쟁 환경의 위협 요인은?
- 시장/수요 변화의 리스크는?
- 규제 강화 가능성은?
- 기술 변화의 위협은?

## TOWS 매트릭스 (전략 도출)

SWOT 분석 결과를 바탕으로 **4가지 전략 방향**을 도출합니다.

```
              │    Strengths (S)    │    Weaknesses (W)
──────────────┼─────────────────────┼─────────────────────
Opportunities │  SO 전략            │  WO 전략
    (O)       │  (강점-기회 활용)     │  (약점 보완-기회 활용)
──────────────┼─────────────────────┼─────────────────────
Threats       │  ST 전략            │  WT 전략
    (T)       │  (강점-위협 대응)     │  (약점-위협 최소화)
```

### SO 전략 (Maxi-Maxi)
- **접근**: 강점을 활용하여 기회 포착
- **질문**: 어떤 강점으로 기회를 극대화할 수 있는가?
- **예시**: 기술력을 바탕으로 신규 시장 진출

### WO 전략 (Mini-Maxi)
- **접근**: 약점을 보완하여 기회 활용
- **질문**: 어떤 약점을 개선해야 기회를 잡을 수 있는가?
- **예시**: 파트너십을 통해 역량 부족 보완

### ST 전략 (Maxi-Mini)
- **접근**: 강점을 활용하여 위협 대응
- **질문**: 어떤 강점으로 위협을 방어할 수 있는가?
- **예시**: 브랜드 파워로 신규 경쟁자 대응

### WT 전략 (Mini-Mini)
- **접근**: 약점과 위협을 최소화
- **질문**: 최악의 시나리오를 어떻게 피할 수 있는가?
- **예시**: 비핵심 사업 철수, 리스크 헷지

## SWOT 분석 프로세스

### Step 1: 데이터 수집
- 내부: 재무, 운영, 인사, 마케팅 데이터
- 외부: 시장, 경쟁, 거시환경(PESTEL) 정보

### Step 2: 요인 식별
- 각 SWOT 영역별 3-5개 핵심 요인 도출
- 구체적, 측정 가능한 표현 사용

### Step 3: 우선순위 결정
- 영향도와 시급성 기준 우선순위 설정

### Step 4: 전략 도출
- TOWS 매트릭스를 활용한 전략 방향 설정

### Step 5: 실행 계획
- 구체적 액션 아이템 도출

## SWOT와 다른 프레임워크 연계

### PESTEL → SWOT
- PESTEL 분석 결과 → Opportunities, Threats

### Porter's Five Forces → SWOT
- 산업 분석 결과 → 경쟁 관련 O/T

### Value Chain → SWOT
- 가치사슬 분석 → Strengths, Weaknesses

## 베스트 프랙티스

### DO
- ✅ 구체적이고 측정 가능한 요인 작성
- ✅ 증거/데이터 기반 분석
- ✅ 다양한 이해관계자 관점 포함
- ✅ 정기적 업데이트

### DON'T
- ❌ 모호하고 일반적인 표현
- ❌ 주관적 의견만으로 판단
- ❌ 너무 많은 요인 나열 (3-5개 권장)
- ❌ 일회성 분석 후 방치

## 한계점

1. **주관성**: 분석자 관점에 따른 편향
2. **정적 분석**: 시점 기준, 동적 변화 미반영
3. **우선순위 부재**: 요인 간 중요도 차이 미반영
4. **전략 연계 약함**: 분석과 실행 사이 갭
