# Porter's Five Forces 프레임워크

## 개요

| 항목 | 내용 |
|------|------|
| **창시자** | Michael E. Porter |
| **소속** | Harvard Business School |
| **발표** | 1979년 |
| **원문** | "How Competitive Forces Shape Strategy", Harvard Business Review |
| **개정** | 2008년, "The Five Competitive Forces That Shape Strategy" |

## 이론적 배경

Porter의 Five Forces는 **산업의 경쟁 구조**를 분석하여 해당 산업의 수익성을 결정하는 요인을 파악하는 프레임워크입니다.

### 핵심 명제
> "산업의 수익성은 5가지 경쟁요인의 강도에 의해 결정된다"

## 5가지 경쟁요인

```
                    [신규 진입자 위협]
                          ↓
[공급자 교섭력] ← [기존 경쟁자 간 경쟁] → [구매자 교섭력]
                          ↑
                    [대체재 위협]
```

### 1. 신규 진입자 위협 (Threat of New Entrants)

**정의**: 새로운 기업이 해당 산업에 진입할 가능성

**진입장벽 요소**:
| 장벽 | 설명 | 분석 질문 |
|------|------|----------|
| 규모의 경제 | 대량 생산의 비용 우위 | 최소 효율 규모는? |
| 제품 차별화 | 브랜드 충성도 | 고객 전환 비용은? |
| 자본 요구량 | 초기 투자 규모 | 진입에 필요한 자본은? |
| 전환 비용 | 고객의 공급자 변경 비용 | 고객 락인 정도는? |
| 유통 채널 | 유통망 접근성 | 기존 채널 장악 정도는? |
| 정부 정책 | 규제, 라이선스 | 법적 진입장벽은? |

### 2. 공급자 교섭력 (Bargaining Power of Suppliers)

**정의**: 공급자가 가격을 올리거나 품질을 낮출 수 있는 능력

**교섭력 강화 요인**:
- 공급자 집중도가 높음
- 대체 공급원이 적음
- 전환 비용이 높음
- 전방 통합 위협 존재
- 구매량이 공급자에게 중요하지 않음

**분석 질문**:
- 핵심 공급자는 몇 개인가?
- 대체 공급자 확보가 가능한가?
- 공급자의 전방 통합 가능성은?

### 3. 구매자 교섭력 (Bargaining Power of Buyers)

**정의**: 구매자가 가격을 낮추거나 품질 향상을 요구할 수 있는 능력

**교섭력 강화 요인**:
- 구매자 집중도가 높음
- 구매량이 큼
- 제품 차별화가 낮음
- 전환 비용이 낮음
- 후방 통합 위협 존재
- 가격 민감도가 높음

**분석 질문**:
- 주요 고객의 구매 비중은?
- 고객의 가격 민감도는?
- 고객의 후방 통합 가능성은?

### 4. 대체재 위협 (Threat of Substitutes)

**정의**: 다른 산업의 제품/서비스가 동일한 기능을 수행할 가능성

**위협 강화 요인**:
- 대체재의 가격 대비 성능이 우수
- 구매자의 전환 비용이 낮음
- 대체재 산업의 수익성이 높음

**분석 질문**:
- 어떤 대체재가 존재하는가?
- 대체재의 가격/성능 비율은?
- 대체재로의 전환 트렌드는?

### 5. 기존 경쟁자 간 경쟁 (Industry Rivalry)

**정의**: 현재 산업 내 기업들 간의 경쟁 강도

**경쟁 강화 요인**:
- 경쟁자 수가 많음
- 산업 성장률이 낮음
- 고정비용이 높음
- 제품 차별화가 낮음
- 퇴출 장벽이 높음
- 전략적 이해관계가 큼

**분석 질문**:
- 주요 경쟁자는 누구인가?
- 시장 점유율 분포는?
- 경쟁의 주요 차원은? (가격, 품질, 서비스)

## 분석 템플릿

### Five Forces 분석표

| 경쟁요인 | 강도 | 주요 근거 | 시사점 |
|----------|------|----------|--------|
| 신규 진입 위협 | 높음/중간/낮음 | | |
| 공급자 교섭력 | 높음/중간/낮음 | | |
| 구매자 교섭력 | 높음/중간/낮음 | | |
| 대체재 위협 | 높음/중간/낮음 | | |
| 기존 경쟁 강도 | 높음/중간/낮음 | | |

### 강도 평가 기준

| 강도 | 설명 | 산업 매력도 |
|------|------|------------|
| **높음** | 해당 요인이 수익성을 크게 감소시킴 | 낮음 |
| **중간** | 해당 요인의 영향이 보통 수준 | 보통 |
| **낮음** | 해당 요인이 수익성에 미치는 영향 미미 | 높음 |

## 전략적 시사점

### 방어적 포지셔닝
경쟁요인이 가장 약한 위치에 포지셔닝

### 경쟁요인 변화
산업 구조를 자사에 유리하게 변화시킴

### 산업 변화 예측
경쟁요인 변화 트렌드를 파악하여 선제 대응

## 한계점

1. **정적 분석**: 특정 시점 스냅샷, 동적 변화 반영 어려움
2. **산업 경계**: 융합 산업에서 경계 정의 어려움
3. **협력 관계**: 경쟁만 강조, 협력/보완관계 미반영
4. **디지털 전환**: 플랫폼/네트워크 효과 미반영
