# 시장규모 추정 방법론 (TAM/SAM/SOM)

## 개요

| 용어 | 정의 |
|------|------|
| **TAM** | Total Addressable Market (전체 시장 규모) |
| **SAM** | Serviceable Addressable Market (접근 가능 시장) |
| **SOM** | Serviceable Obtainable Market (확보 가능 시장) |

## 시장규모 계층 구조

```
┌─────────────────────────────────────┐
│              TAM                     │
│    (전체 시장 - 모든 잠재 고객)        │
│  ┌─────────────────────────────┐    │
│  │           SAM               │    │
│  │  (접근 가능 - 타겟 세그먼트)  │    │
│  │  ┌─────────────────────┐   │    │
│  │  │        SOM         │   │    │
│  │  │  (확보 가능 - 현실적)│   │    │
│  │  └─────────────────────┘   │    │
│  └─────────────────────────────┘    │
└─────────────────────────────────────┘
```

## TAM (Total Addressable Market)

### 정의
제품/서비스가 도달할 수 있는 **전체 시장의 총 수요**

### 용도
- 시장의 전체 기회 규모 파악
- 투자자 대상 시장 잠재력 제시
- 장기 성장 가능성 평가

### 계산 방법

#### 1. Top-Down 방식

**접근법**: 전체 시장 → 세분화

```
TAM = 산업 전체 시장규모 × 관련 비율

예시: 글로벌 CRM 소프트웨어 시장
- 글로벌 기업용 소프트웨어 시장: $500B
- CRM 비중: 10%
- TAM = $500B × 10% = $50B
```

**장점**: 빠른 추정, 산업 보고서 활용
**단점**: 정확도 낮음, 가정에 의존

#### 2. Bottom-Up 방식

**접근법**: 단위 → 집계

```
TAM = 잠재 고객 수 × 연간 구매액

예시: B2B SaaS 제품
- 전 세계 타겟 기업 수: 10만개
- 연간 구독료: $10,000
- TAM = 100,000 × $10,000 = $1B
```

**장점**: 논리적, 검증 가능
**단점**: 시간 소요, 데이터 필요

#### 3. Value Theory 방식

**접근법**: 가치 기반 추정

```
TAM = 고객 가치 × 적정 가격 비율 × 고객 수

예시: 생산성 도구
- 시간 절약 가치: 연 $50,000/인
- 적정 가격 비율: 10%
- 타겟 사용자: 100만명
- TAM = $50,000 × 10% × 1M = $5B
```

## SAM (Serviceable Addressable Market)

### 정의
TAM 중 **실제로 서비스 가능한** 시장 범위

### 제약 요인
- 지역적 범위 (진출 국가)
- 고객 세그먼트 (타겟 고객층)
- 제품 기능 (기술적 제약)
- 유통 채널 (판매 방식)

### 계산 방법

```
SAM = TAM × 서비스 가능 비율

예시: 미국 시장 진출 SaaS
- TAM (글로벌): $1B
- 미국 시장 비중: 40%
- 타겟 세그먼트 비중: 50%
- SAM = $1B × 40% × 50% = $200M
```

### SAM 산출 체크리스트

- [ ] 진출 지역 범위 정의
- [ ] 타겟 고객 세그먼트 정의
- [ ] 제품/서비스 적용 범위 정의
- [ ] 유통 채널 제약 고려

## SOM (Serviceable Obtainable Market)

### 정의
SAM 중 **현실적으로 확보 가능한** 시장 점유율

### 결정 요인
- 경쟁 강도
- 마케팅/영업 역량
- 자원 제약 (자금, 인력)
- 시장 진입 시점

### 계산 방법

```
SOM = SAM × 예상 시장점유율

예시: 신규 진입 스타트업
- SAM: $200M
- 3년차 예상 점유율: 5%
- SOM = $200M × 5% = $10M
```

### 시장점유율 추정 방법

#### 1. 경쟁사 벤치마킹
```
유사 기업의 시장점유율 참고
- 동일 산업 스타트업 평균: 2-5% (3년차)
- 유니콘 기업 평균: 10-15% (5년차)
```

#### 2. 고객 확보 모델
```
SOM = 예상 고객 수 × 평균 계약 가치

- 연간 신규 고객: 500개사
- 평균 계약 가치: $20,000
- SOM = 500 × $20,000 = $10M
```

#### 3. 성장률 기반
```
SOM = 현재 매출 × (1 + 성장률)^n

- 현재 매출: $1M
- 연간 성장률: 100%
- 3년 후 SOM = $1M × (1+1)^3 = $8M
```

## 시장규모 검증 방법

### 삼각측량 (Triangulation)
```
Top-Down 결과와 Bottom-Up 결과를 비교하여 검증

- Top-Down TAM: $1.2B
- Bottom-Up TAM: $0.9B
- 조정 후 TAM: $1.0B (중간값 또는 보수적 추정)
```

### 산업 보고서 비교
```
공신력 있는 기관의 시장규모 추정과 비교

출처 예시:
- Gartner
- IDC
- McKinsey
- Statista
- IBISWorld
```

### 전문가 인터뷰
```
산업 전문가의 시장규모 의견 수렴
- 규모의 합리성 검토
- 가정의 타당성 확인
```

## 투자자 관점 시장규모

### 벤처캐피탈 기준

| 규모 | 해석 |
|------|------|
| TAM < $1B | 니치 시장, 규모 투자 어려움 |
| TAM $1-10B | 적정 규모, 시리즈 A-B 투자 가능 |
| TAM > $10B | 대형 시장, 유니콘 가능성 |

### 피칭 시 주의사항

#### DO
- ✅ 명확한 산출 근거 제시
- ✅ Top-Down + Bottom-Up 모두 제시
- ✅ 가정 명시
- ✅ 출처 표기

#### DON'T
- ❌ 지나치게 낙관적 추정
- ❌ 근거 없는 수치
- ❌ TAM만 강조 (SAM/SOM 무시)
- ❌ 성장률 과대 추정

## 시장 성장률 추정

### CAGR (연평균 성장률)

```
CAGR = (최종연도 시장규모 / 기준연도 시장규모)^(1/n) - 1

예시:
- 2020년: $10B
- 2025년: $20B
- CAGR = ($20B / $10B)^(1/5) - 1 = 14.9%
```

### 성장률 출처
- 산업 보고서 (Gartner, IDC)
- 과거 트렌드 분석
- 전문가 전망

## 한계점 및 주의사항

1. **가정 의존성**: 추정의 정확도는 가정에 의존
2. **시점 한계**: 시장은 동적으로 변화
3. **경계 모호성**: 시장 경계 정의의 어려움
4. **데이터 한계**: 정확한 데이터 확보의 어려움
