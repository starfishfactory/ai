# ISO 20252:2019 시장조사 품질 표준

## 개요

| 항목 | 내용 |
|------|------|
| **표준명** | ISO 20252:2019 |
| **정식 명칭** | Market, opinion and social research, including insights and data analytics — Vocabulary and service requirements |
| **관리 기관** | ISO/TC 225 |
| **최초 발행** | 2006년 |
| **현행 버전** | 2019년 (3rd edition) |
| **다음 개정** | 2026년 예정 |

## 표준의 목적

ISO 20252는 시장조사, 여론조사, 사회조사 프로젝트의 **품질 관리 체계**를 정의합니다.

### 적용 범위
- 조사 계획 수립
- 데이터 수집 (정량/정성)
- 데이터 처리 및 분석
- 결과 보고

## 핵심 원칙

### 1. 투명성 (Transparency)
- 조사 방법론 명시
- 표본 설계 근거 제시
- 한계점 명확히 기술

### 2. 재현성 (Reproducibility)
- 동일 조건에서 유사 결과 도출 가능
- 조사 절차 문서화

### 3. 윤리성 (Ethics)
- 응답자 프라이버시 보호
- 데이터 기밀 유지
- 이해충돌 방지

## 조사 프로세스 요건

### Phase 1: 조사 설계
```
1. 조사 목적 정의
2. 조사 범위 설정
3. 방법론 선택 (정량/정성)
4. 표본 설계
5. 조사 도구 개발 (설문지 등)
```

### Phase 2: 데이터 수집
```
1. 현장 조사 실시
2. 품질 모니터링
3. 응답 데이터 검증
4. 조사원 관리
```

### Phase 3: 데이터 처리
```
1. 코딩 및 입력
2. 데이터 정제
3. 가중치 적용
4. 통계 분석
```

### Phase 4: 보고
```
1. 분석 결과 해석
2. 보고서 작성
3. 한계점 명시
4. 권고사항 제시
```

## 품질 관리 체크리스트

### 조사 설계 단계
- [ ] 조사 목적이 명확히 정의되었는가?
- [ ] 표본 크기가 통계적으로 적절한가?
- [ ] 표본 추출 방법이 적합한가?
- [ ] 조사 도구가 검증되었는가?

### 데이터 수집 단계
- [ ] 조사원 교육이 실시되었는가?
- [ ] 품질 모니터링 체계가 있는가?
- [ ] 응답률이 적정 수준인가?
- [ ] 비응답 편향이 관리되고 있는가?

### 분석 및 보고 단계
- [ ] 분석 방법이 목적에 적합한가?
- [ ] 결과 해석이 데이터에 근거하는가?
- [ ] 한계점이 명시되었는가?
- [ ] 일반화 가능 범위가 명확한가?

## 시장분석 적용 지침

### 1차 자료 수집 시
- 표본 설계 근거 명시
- 조사 방법 상세 기술
- 응답률 및 표본 특성 보고

### 2차 자료 활용 시
- 출처 명시 (저자, 발행일, 기관)
- 원 데이터의 수집 방법 확인
- 데이터 시점 및 범위 명시

### 분석 결과 보고 시
- 방법론 섹션 필수 포함
- 가정 및 한계점 명시
- 신뢰구간/오차범위 표기
