# PESTEL 분석 프레임워크

## 개요

| 항목 | 내용 |
|------|------|
| **창시자** | Francis J. Aguilar |
| **소속** | Harvard Business School |
| **발표** | 1967년 |
| **원문** | "Scanning the Business Environment" |
| **원래 명칭** | ETPS → STEP → STEEP → PESTEL |

## 이론적 배경

PESTEL은 **거시환경(Macro Environment)**을 체계적으로 분석하는 프레임워크입니다.

### 명칭 변화
```
1967: ETPS (Economic, Technical, Political, Social) - Aguilar
1970s: STEP (Strategic Trend Evaluation Process) - Arnold Brown
1980s: STEEP (+ Environmental)
1980s~: PESTEL/PESTLE (+ Legal)
```

## 6가지 환경요인

```
┌─────────────────────────────────────────────┐
│              거시환경 (PESTEL)               │
├─────────┬─────────┬─────────┬───────────────┤
│Political│Economic │ Social  │ Technological │
├─────────┼─────────┼─────────┼───────────────┤
│Environmental      │  Legal                  │
└───────────────────┴─────────────────────────┘
                    ↓
              [산업/기업]
```

### 1. Political (정치적 요인)

**정의**: 정부 정책, 정치적 안정성이 사업에 미치는 영향

| 분석 항목 | 세부 내용 |
|----------|----------|
| 정부 안정성 | 정권 교체 가능성, 정치 리스크 |
| 조세 정책 | 법인세, 관세, 세금 인센티브 |
| 무역 규제 | 수출입 제한, 무역 협정 |
| 정부 지출 | 공공 투자, 보조금 정책 |
| 노동법 | 최저임금, 근로시간 규제 |

**분석 질문**:
- 현 정부의 산업 정책 기조는?
- 규제 강화/완화 방향은?
- 정치적 리스크 수준은?

### 2. Economic (경제적 요인)

**정의**: 거시경제 상황이 사업에 미치는 영향

| 분석 항목 | 세부 내용 |
|----------|----------|
| GDP 성장률 | 경제 성장 추세 |
| 인플레이션 | 물가 상승률, 구매력 변화 |
| 금리 | 대출 비용, 투자 수익률 |
| 환율 | 수출입 경쟁력, 환리스크 |
| 실업률 | 노동시장 상황, 소비 영향 |
| 가처분소득 | 소비자 구매력 |

**분석 질문**:
- 경기 사이클 어느 단계인가?
- 소비자 구매력 추세는?
- 환율/금리 전망은?

### 3. Social (사회적 요인)

**정의**: 인구통계, 문화, 라이프스타일 변화가 사업에 미치는 영향

| 분석 항목 | 세부 내용 |
|----------|----------|
| 인구 구조 | 고령화, 출산율, 인구 성장 |
| 소득 분포 | 빈부격차, 중산층 규모 |
| 라이프스타일 | 가치관, 소비 패턴 변화 |
| 교육 수준 | 학력, 기술 인력 공급 |
| 문화적 태도 | 사회적 규범, 트렌드 |
| 건강 의식 | 웰빙, 환경 의식 |

**분석 질문**:
- 주요 인구통계 변화는?
- 소비자 가치관 변화는?
- 사회적 트렌드는?

### 4. Technological (기술적 요인)

**정의**: 기술 발전이 사업에 미치는 영향

| 분석 항목 | 세부 내용 |
|----------|----------|
| 기술 혁신 | 신기술 등장, R&D 동향 |
| 자동화 | 생산성 향상, 고용 영향 |
| 디지털 전환 | 온라인화, 플랫폼 경제 |
| 기술 인프라 | 통신, 인터넷 보급률 |
| 기술 수명주기 | 기술 진부화 속도 |
| 지적재산권 | 특허, 기술 보호 |

**분석 질문**:
- 산업을 변화시킬 핵심 기술은?
- 기술 진입장벽은?
- 디지털 전환 영향은?

### 5. Environmental (환경적 요인)

**정의**: 자연환경, 기후변화가 사업에 미치는 영향

| 분석 항목 | 세부 내용 |
|----------|----------|
| 기후변화 | 온난화, 기상이변 |
| 환경 규제 | 탄소배출, 폐기물 규제 |
| 지속가능성 | ESG, 친환경 요구 |
| 자원 가용성 | 원자재, 에너지 공급 |
| 생태계 영향 | 생물다양성, 오염 |

**분석 질문**:
- 환경 규제 강화 방향은?
- 기후변화 사업 영향은?
- ESG 요구 수준은?

### 6. Legal (법적 요인)

**정의**: 법률, 규제가 사업에 미치는 영향

| 분석 항목 | 세부 내용 |
|----------|----------|
| 산업 규제 | 인허가, 영업 규제 |
| 소비자 보호 | 제조물 책임, 광고 규제 |
| 고용 법규 | 차별 금지, 안전 규정 |
| 경쟁법 | 독점 규제, 공정거래 |
| 데이터 보호 | 개인정보, 프라이버시 |
| 지적재산권 | 특허, 상표, 저작권 |

**분석 질문**:
- 핵심 규제 요건은?
- 법규 변화 방향은?
- 규제 준수 비용은?

## 분석 템플릿

### PESTEL 분석표

| 요인 | 현황 | 트렌드 | 영향도 | 시사점 |
|------|------|--------|--------|--------|
| **P** Political | | ↑/→/↓ | 상/중/하 | |
| **E** Economic | | ↑/→/↓ | 상/중/하 | |
| **S** Social | | ↑/→/↓ | 상/중/하 | |
| **T** Technological | | ↑/→/↓ | 상/중/하 | |
| **E** Environmental | | ↑/→/↓ | 상/중/하 | |
| **L** Legal | | ↑/→/↓ | 상/중/하 | |

### 영향도 매트릭스

```
           높은 영향
              ↑
    [중요/모니터링] │ [최우선 관리]
                   │
  ←───────────────┼───────────────→ 높은 불확실성
                   │
    [낮은 우선순위] │ [시나리오 분석]
              ↓
           낮은 영향
```

## 분석 프로세스

### Step 1: 환경 스캐닝
각 PESTEL 요인별 현황 파악

### Step 2: 트렌드 식별
변화 방향과 속도 파악

### Step 3: 영향 평가
사업에 미치는 영향도 평가

### Step 4: 시사점 도출
전략적 시사점 및 대응 방안

## PESTEL과 다른 프레임워크 연계

### PESTEL → SWOT
- PESTEL의 기회/위협 → SWOT의 O/T

### PESTEL → Porter's Five Forces
- 거시환경 변화 → 산업 경쟁구조 변화

### PESTEL → 시나리오 플래닝
- 불확실성 높은 요인 → 시나리오 변수

## 한계점

1. **일반화 위험**: 산업/기업 특성 반영 부족
2. **상호작용 미반영**: 요인 간 연관성 파악 어려움
3. **동적 변화**: 정적 분석의 한계
4. **주관성**: 분석자 관점에 따른 편향
