# PESTEL 분석 프레임워크

## 출처
- **창시자**: Francis J. Aguilar
- **기관**: Harvard Business School
- **연도**: 1967년
- **원문**: "Scanning the Business Environment"

## 6가지 요인

| 요인 | 핵심 항목 |
|------|----------|
| **P**olitical | 정부정책, 조세, 무역규제 |
| **E**conomic | GDP, 인플레이션, 금리, 환율 |
| **S**ocial | 인구구조, 라이프스타일, 가치관 |
| **T**echnological | 기술혁신, 디지털전환, R&D |
| **E**nvironmental | 기후변화, ESG, 환경규제 |
| **L**egal | 산업규제, 소비자보호, 데이터보호 |

## 분석 흐름
```
거시환경 스캔 → 트렌드 식별 → 영향 평가 → 시사점 도출
```

## 연계
PESTEL → SWOT의 O/T로 연결
