# Consulting Frameworks Reference

## Key Frameworks
| Framework | Developer | Year | Purpose |
|-----------|-----------|------|---------|
| BCG Matrix | Boston Consulting Group | 1970 | Portfolio analysis |
| GE/McKinsey Matrix | McKinsey & Company | 1970s | Business unit prioritization |
| McKinsey 7S | McKinsey & Company | 1980s | Organizational alignment |
| Value Chain | Michael Porter | 1985 | Value creation analysis |

## BCG Matrix
```
Stars | Question Marks
------+----------------
Cash Cows | Dogs

Axes: Market Growth Rate (Y) x Relative Market Share (X)
```

## GE Matrix (9-Box)
```
Industry Attractiveness (H/M/L) x Competitive Strength (S/M/W) -> Invest/Select/Divest
```

## 7S Framework
Hard S: Strategy, Structure, Systems
Soft S: Shared Values, Style, Staff, Skills

## Value Chain
- Primary Activities: Inbound Logistics -> Operations -> Outbound Logistics -> Marketing -> Service
- Support Activities: Infrastructure, HR, Technology Development, Procurement
