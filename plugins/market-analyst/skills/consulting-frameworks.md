# 컨설팅 회사 분석 프레임워크

## 개요

세계적 컨설팅 회사들이 개발한 전략 분석 프레임워크입니다.

| 프레임워크 | 개발사 | 연도 | 용도 |
|------------|--------|------|------|
| BCG Growth-Share Matrix | Boston Consulting Group | 1970 | 포트폴리오 분석 |
| GE/McKinsey Matrix | McKinsey & Company | 1970s | 사업부 투자 우선순위 |
| McKinsey 7S | McKinsey & Company | 1980s | 조직 정렬 분석 |
| Value Chain | Michael Porter | 1985 | 가치 창출 분석 |

---

## 1. BCG Growth-Share Matrix

### 배경

| 항목 | 내용 |
|------|------|
| **개발** | Boston Consulting Group |
| **창시자** | Bruce Henderson |
| **연도** | 1970년 |
| **목적** | 다각화 기업의 사업 포트폴리오 분석 |

### 매트릭스 구조

```
         높은 시장 성장률
              ↑
    ┌─────────┬─────────┐
    │   ⭐    │   ❓    │
    │  Stars  │Question │
    │         │  Marks  │
    ├─────────┼─────────┤
    │   🐄    │   🐕    │
    │  Cash   │  Dogs   │
    │  Cows   │         │
    └─────────┴─────────┘
낮은 점유율 ←───────────→ 높은 점유율

         낮은 시장 성장률
```

### 4가지 사분면

#### ⭐ Stars (스타)
| 특성 | 전략 |
|------|------|
| 높은 성장률 + 높은 점유율 | 투자 지속 |
| 시장 리더 위치 | 점유율 유지/확대 |
| 현금 창출 = 투자 소요 | 향후 Cash Cow 전환 목표 |

#### ❓ Question Marks (물음표)
| 특성 | 전략 |
|------|------|
| 높은 성장률 + 낮은 점유율 | 선별적 투자 |
| 성장 잠재력 있음 | Star 전환 또는 철수 결정 |
| 현금 소모 > 창출 | 집중 투자 또는 포기 |

#### 🐄 Cash Cows (현금소)
| 특성 | 전략 |
|------|------|
| 낮은 성장률 + 높은 점유율 | 수확 (Harvest) |
| 안정적 현금 창출 | 최소 투자, 이익 극대화 |
| 성숙 시장 리더 | 다른 사업부에 자금 지원 |

#### 🐕 Dogs (개)
| 특성 | 전략 |
|------|------|
| 낮은 성장률 + 낮은 점유율 | 철수/매각 |
| 현금 함정 위험 | 정리 또는 니치 집중 |
| 전략적 가치 낮음 | 자원 재배치 |

### 분석 템플릿

| 사업부/제품 | 시장 성장률 | 상대적 점유율 | 분류 | 전략 권고 |
|------------|------------|--------------|------|----------|
| | 높/중/낮 | 높/중/낮 | ⭐/❓/🐄/🐕 | |

### 한계점
- 시장 정의의 어려움
- 2개 변수만 고려 (단순화)
- 시너지 효과 미반영
- 현금흐름 가정의 한계

## 2. GE/McKinsey 9-Box Matrix

### 배경

| 항목 | 내용 |
|------|------|
| **개발** | McKinsey & Company (for GE) |
| **연도** | 1970년대 |
| **목적** | BCG 매트릭스의 한계 보완 |
| **특징** | 다차원 변수 고려 |

### 매트릭스 구조

```
산업 매력도
   높음   ┌─────────┬─────────┬─────────┐
          │ 투자/   │ 선별적  │ 선별적  │
          │ 성장    │ 투자    │ 투자    │
          ├─────────┼─────────┼─────────┤
   중간   │ 선별적  │ 선별/   │ 수확/   │
          │ 투자    │ 수익    │ 철수    │
          ├─────────┼─────────┼─────────┤
   낮음   │ 선별적  │ 수확/   │ 철수/   │
          │ 투자    │ 철수    │ 매각    │
          └─────────┴─────────┴─────────┘
              강함      중간       약함
                    경쟁 강점
```

### 평가 요소

#### 산업 매력도 (Industry Attractiveness)

| 요소 | 가중치 예시 |
|------|------------|
| 시장 규모 | 20% |
| 시장 성장률 | 20% |
| 수익성 | 15% |
| 경쟁 강도 | 15% |
| 기술 요구 수준 | 10% |
| 환경/규제 영향 | 10% |
| 진입 장벽 | 10% |

#### 경쟁 강점 (Competitive Strength)

| 요소 | 가중치 예시 |
|------|------------|
| 시장 점유율 | 20% |
| 점유율 성장 | 15% |
| 제품 품질 | 15% |
| 브랜드 인지도 | 15% |
| 유통 네트워크 | 10% |
| 생산 역량 | 10% |
| R&D 역량 | 15% |

### 점수 산출 방법

```
1. 각 요소별 가중치 설정 (합계 100%)
2. 각 요소별 점수 부여 (1-5점)
3. 가중 평균 계산

예시:
산업 매력도 = (20%×4) + (20%×5) + (15%×3) + ... = 3.8
경쟁 강점 = (20%×4) + (15%×3) + (15%×4) + ... = 3.5
```

### 전략 권고

| 위치 | 전략 | 자원 배분 |
|------|------|----------|
| 녹색 (투자/성장) | 공격적 투자, 시장 리더십 추구 | 높음 |
| 황색 (선별적) | 선택적 투자, 세그먼트 집중 | 중간 |
| 적색 (수확/철수) | 현금 회수, 철수 검토 | 낮음 |

### 분석 템플릿

| 사업부 | 산업 매력도 | 경쟁 강점 | 위치 | 전략 |
|--------|------------|----------|------|------|
| | (1-5) | (1-5) | 녹/황/적 | |

## 3. McKinsey 7S Framework

### 배경

| 항목 | 내용 |
|------|------|
| **개발** | McKinsey & Company |
| **창시자** | Tom Peters, Robert Waterman |
| **연도** | 1980년대 |
| **원문** | "In Search of Excellence" (1982) |
| **목적** | 조직 효과성 분석 |

### 7S 구조

```
              ┌─────────┐
              │ Strategy│
              └────┬────┘
                   │
        ┌──────────┼──────────┐
        │          │          │
   ┌────┴────┐ ┌───┴───┐ ┌────┴────┐
   │Structure│ │Systems│ │  Style  │
   └────┬────┘ └───┬───┘ └────┬────┘
        │          │          │
        └──────────┼──────────┘
                   │
         ┌─────────┴─────────┐
         │   Shared Values   │ (중심)
         └─────────┬─────────┘
                   │
        ┌──────────┼──────────┐
        │          │          │
   ┌────┴────┐ ┌───┴───┐
   │  Staff  │ │Skills │
   └─────────┘ └───────┘
```

### 7가지 요소

#### Hard S (구조적, 변경 용이)

| 요소 | 정의 | 분석 질문 |
|------|------|----------|
| **Strategy** | 경쟁 우위 달성 계획 | 전략 방향은? 차별화 요소는? |
| **Structure** | 조직 구조 | 보고 체계는? 의사결정 구조는? |
| **Systems** | 업무 프로세스/절차 | 핵심 시스템은? IT 인프라는? |

#### Soft S (문화적, 변경 어려움)

| 요소 | 정의 | 분석 질문 |
|------|------|----------|
| **Shared Values** | 핵심 가치/문화 | 조직의 핵심 가치는? 문화는? |
| **Style** | 리더십 스타일 | 경영진 리더십은? 의사결정 방식은? |
| **Staff** | 인력/인재 | 핵심 인력은? 인재 확보 방법은? |
| **Skills** | 핵심 역량 | 차별화 역량은? 역량 개발 방향은? |

### 활용 방법

#### 1. 현황 진단
각 S 요소의 현재 상태 평가

#### 2. 정렬 분석
7S 간 일관성/정렬 수준 평가

```
정렬 매트릭스:
        Strategy Structure Systems Style Staff Skills
Strategy    -       ?        ?      ?     ?      ?
Structure   ?       -        ?      ?     ?      ?
...
(? = 정렬 수준: ○ 높음, △ 보통, × 낮음)
```

#### 3. 갭 분석
현재 상태 vs 목표 상태 비교

#### 4. 변화 계획
정렬 개선을 위한 액션 플랜

### 분석 템플릿

| 요소 | 현재 상태 | 목표 상태 | 갭 | 액션 |
|------|----------|----------|-----|------|
| Strategy | | | | |
| Structure | | | | |
| Systems | | | | |
| Shared Values | | | | |
| Style | | | | |
| Staff | | | | |
| Skills | | | | |

## 4. Value Chain Analysis

### 배경

| 항목 | 내용 |
|------|------|
| **창시자** | Michael E. Porter |
| **연도** | 1985년 |
| **원문** | "Competitive Advantage" |
| **목적** | 가치 창출 활동 분석 |

### 가치사슬 구조

```
┌─────────────────────────────────────────────────────────┐
│                    지원 활동 (Support Activities)         │
├─────────────────────────────────────────────────────────┤
│ 기업 인프라 (Firm Infrastructure)                        │
├─────────────────────────────────────────────────────────┤
│ 인적자원 관리 (Human Resource Management)                │
├─────────────────────────────────────────────────────────┤
│ 기술 개발 (Technology Development)                       │
├─────────────────────────────────────────────────────────┤
│ 조달 (Procurement)                                       │
├─────────────────────────────────────────────────────────┤
│                    본원 활동 (Primary Activities)         │
├─────────┬─────────┬─────────┬─────────┬─────────────────┤
│ 내부    │ 운영    │ 외부    │ 마케팅  │ 서비스          │
│ 물류    │(Oper-  │ 물류    │ & 영업  │ (Service)       │
│(Inbound│ations) │(Outbound│(Marketing│                 │
│Logistics│        │Logistics│& Sales) │                 │ → 이익
└─────────┴─────────┴─────────┴─────────┴─────────────────┘
```

### 본원 활동

| 활동 | 내용 | 분석 포인트 |
|------|------|------------|
| **내부 물류** | 원자재 수령, 저장, 배분 | 재고 관리, 공급망 |
| **운영** | 투입물의 제품 전환 | 생산 효율성, 품질 |
| **외부 물류** | 제품 저장, 유통 | 배송 속도, 비용 |
| **마케팅/영업** | 구매 유도 활동 | 브랜드, 채널, 가격 |
| **서비스** | 제품 가치 유지/향상 | 고객 지원, A/S |

### 지원 활동

| 활동 | 내용 | 분석 포인트 |
|------|------|------------|
| **기업 인프라** | 경영, 재무, 법무 | 의사결정, 거버넌스 |
| **인적자원 관리** | 채용, 교육, 보상 | 인재 확보, 역량 개발 |
| **기술 개발** | R&D, 프로세스 혁신 | 혁신 역량, 특허 |
| **조달** | 투입물 구매 | 공급자 관리, 비용 |

### 분석 템플릿

| 활동 | 비용 비중 | 차별화 기여 | 경쟁사 대비 | 개선 기회 |
|------|----------|------------|------------|----------|
| 내부 물류 | % | 상/중/하 | 우위/동등/열위 | |
| 운영 | % | 상/중/하 | 우위/동등/열위 | |
| 외부 물류 | % | 상/중/하 | 우위/동등/열위 | |
| 마케팅/영업 | % | 상/중/하 | 우위/동등/열위 | |
| 서비스 | % | 상/중/하 | 우위/동등/열위 | |
