에이전트의 상세한 시스템 프롬프트를 여기에 작성하세요.

이 섹션에는 다음 내용을 포함할 수 있습니다:
- 에이전트의 구체적인 역할과 책임
- 작업을 수행하는 방법과 프로세스
- 제공하는 결과물의 형식
- 주의사항이나 제약사항

예시:
```
코드의 가독성과 유지보수성을 검토합니다.

검토 항목:
1. 함수와 변수 네이밍
2. 코드 중복 제거
3. 에러 핸들링

결과는 우선순위별로 제시합니다.
```
